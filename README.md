# QuanLySach
# Hệ Thống Quản Lý Thư Viện Sách (C)

Đồ án môn học Cấu trúc dữ liệu và Giải thuật.

## Chức năng

* Quản lý danh sách sách bằng **Danh sách liên kết đơn (Linked List)** (không giới hạn số lượng).
* Thêm, xóa, chỉnh sửa thông tin sách (hỗ trợ nhập tên sách có dấu cách).
* Tìm kiếm sách nhanh theo Mã số sách (ID).
* Sắp xếp sách tự động theo thứ tự tăng dần bằng thuật toán **Radix Sort**.
* Đọc/Ghi dữ liệu file text (tự động lưu `data.txt` khi thoát, xuất báo cáo `output.txt`).
* Quản lý bộ nhớ động thủ công (cấp phát/giải phóng RAM).

## Ngôn ngữ sử dụng

* Ngôn ngữ: C
* Kỹ thuật: Con trỏ (Pointer), Struct, Linked List, Radix Sort, File I/O.
